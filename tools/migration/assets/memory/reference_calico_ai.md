Calico AI (heycalico.ai) — AI creative production platform.

**What it does:** AI-generated video ads, UGC-style content, image ads, property videos. 130+ AI avatars, 35+ languages, up to 4K resolution.

**Pricing:**
- Lite: $49/mo (200 credits, NO API)
- Starter: $99/mo (450 credits, NO API)
- Pro: $199/mo (950 credits, API access)
- Scale: $399/mo (2,000 credits, API access, dedicated success manager)
- Yearly saves 20%

**API access starts at Pro ($199/mo).** Credits are used per generation.

**Relevance to BlueMarlin:** Interesting for future development — could auto-generate video ads and image content for clients' social media. Not actionable now but worth keeping in mind when the content pipeline matures. The user finds this genuinely interesting.

**Not a replacement for image generation APIs** — for programmatic social media post images, OpenAI/Imagen/Stability are better fits at $0.01-0.17/image. Calico is for higher-production video/ad content.
